# 한 장으로 끝내는 대한민국 제도 100

대한민국의 주요 제도를 한 장의 구조도로 읽는 책 프로젝트입니다.

## 핵심 콘셉트

기업에 비즈니스 모델이 있듯이, 국가에는 제도 모델이 있다.

이 프로젝트는 법령, 조직, 절차, 예산, 문서, 데이터, 권한 관계를 하나의 표준 포맷으로 정리해 `제도가 실제로 어떻게 작동하는지` 보여주는 것을 목표로 합니다.
